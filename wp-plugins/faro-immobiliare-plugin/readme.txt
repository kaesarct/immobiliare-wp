Plugin per la gestione di annunci immobiliari con custom post type, metabox e REST API.

== Description ==

Faro Immobiliare Plugin è un plugin completo per la gestione di annunci immobiliari su WordPress.

**Caratteristiche principali:**

* Custom Post Type "Proprietà" con supporto completo
* Tassonomie per tipologie e località
* Metabox avanzati per dettagli proprietà (prezzo, mq, camere, bagni, indirizzo, stato)
* Pannello amministrativo personalizzato
* Colonne personalizzate nella lista proprietà
* REST API per integrazione frontend
* Completamente tradotto in italiano
* Sicuro e ottimizzato per le performance

**Campi disponibili:**
* Prezzo
* Metri quadri
* Numero camere
* Numero bagni
* Indirizzo
* Stato (vendita/affitto/venduto)

== Installation ==

1. Carica la cartella del plugin nella directory `/wp-content/plugins/`
2. Attiva il plugin attraverso il menu 'Plugin' in WordPress
3. Vai su "Immobili" nel menu amministrativo per iniziare ad aggiungere proprietà
4. Configura le impostazioni in "Immobili > Impostazioni"

== Frequently Asked Questions ==

= Come aggiungo una nuova proprietà? =

Vai su "Immobili > Aggiungi Proprietà" nel pannello amministrativo e compila tutti i campi necessari.

= Posso personalizzare i campi? =

Sì, il plugin è completamente personalizzabile attraverso hook e filtri WordPress.

= È compatibile con il mio tema? =

Sì, il plugin utilizza i template standard di WordPress e si integra con qualsiasi tema.
